Typographie

La propriété font-family permet de définir la police de caractères utilisée pour le texte d'un élément. Par exemple :

				p {
					font-family: Arial, sans-serif;
				}

La propriété font-size permet de définir la taille de la police de caractères. Par exemple :

				h1 {
					font-size: 2em;
				}

La propriété line-height permet de définir l'espacement entre les lignes de texte. Par exemple :

				p {
					line-height: 1.5;
				}

Couleurs

La propriété color permet de définir la couleur du texte. Par exemple :

				p {
					color: #333;
				}

La propriété background-color permet de définir la couleur de fond d'un élément. Par exemple :

				body {
					background-color: #fff;
				}

Bordures

La propriété border permet de définir une bordure autour d'un élément. Par exemple :

				div {
					border: 1px solid #333;
				}

Ombres

La propriété box-shadow permet de définir une ombre autour d'un élément. Par exemple :

				div {
					box-shadow: 0 0 10px 
                                        rgba(0, 0, 0, 0.5);
				}

Media Queries

Les media queries permettent de modifier le style d'un site web en fonction de la taille de l'écran du dispositif utilisé pour le visualiser. Par exemple :

				@media (max-width: 600px) {
					div {
						width: 100%;
					}
				}
